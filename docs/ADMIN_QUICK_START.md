## âš¡ Quick Actions

### Add New Product
1. Click **"Add Product"**
2. Fill in **Name**, **Slug**, **Description**, and **Price**
3. Upload **Images** (1200x1200px)
4. Select **Category**, **Materials**, and **Tags**
5. Set **Status** to "published"
6. Set **In Stock**, **Availability**, and **Lead Time**
7. Click **"Create Product"**

### Edit Product
1. Find product in grid
2. Click **pencil icon**
3. Make changes
4. Click **"Update Product"**

### Mark Out of Stock
1. Edit product
2. Toggle **"In Stock"** OFF
3. Save

### Feature Product
1. Edit product
2. Toggle **"Featured"** ON
3. Save
