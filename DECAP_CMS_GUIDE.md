## üì¶ **Adding Products**

### **Step-by-Step Process:**

#### **1. Create New Product**
1. In the CMS dashboard, click **"Products"**
2. Click the **"New Product"** button
3. Fill in the product details

#### **2. Required Information**
- **Product ID**: Unique code (e.g., `ring-gold-001`)
- **URL Slug**: Web address ending (auto-generated)
- **Product Name**: Full display name
- **Price (LKR)**: Price in Sri Lankan Rupees
- **Product Images**: Upload 1-5 photos
- **Category**: Choose from dropdown
- **Product Description**: Full details in rich text

#### **3. Optional Information**
- **SKU**: Internal product code
- **Materials**: List what it's made of
- **Weight**: In grams
- **Dimensions**: Size measurements
- **Available Sizes**: For rings/bracelets
- **Tags**: Search keywords

#### **4. Upload Images**
- **Formats**: JPG, PNG, WebP
- **Size**: Up to 5MB per image
- **Quality**: High-resolution product photos
- **Quantity**: 1-5 images per product

#### **5. Save and Publish**
- **Save**: Creates a draft (not live)
- **Publish**: Makes the product live on your website
- **Editorial Workflow**: Draft ‚Üí Review ‚Üí Publish

## üîß **How It Works**

### **Behind the Scenes:**
1. **You create/edit products** in the CMS
2. **Decap saves changes** to GitHub automatically  
3. **GitHub triggers a rebuild** of your website
4. **Your products appear live** within 5-10 minutes

### **Editorial Workflow:**
- **Draft**: Product is saved but not published
- **In Review**: Ready for approval (if you have reviewers)
- **Published**: Live on your website

## üÜò **Troubleshooting**

### **Common Issues:**

#### **Can't Login?**
- Check your email for invitation
- Ensure you're using the correct URL
- Contact admin if account isn't set up

#### **Images Won't Upload?**
- Check file size (must be under 5MB)
- Use JPG or PNG format
- Ensure good internet connection

#### **Product Not Appearing?**
- Wait 5-10 minutes for site rebuild
- Check if product is published (not just saved)
- Refresh your website page

#### **Lost Changes?**
- All work is automatically saved
- Check drafts section in CMS
- Previous versions are backed up
